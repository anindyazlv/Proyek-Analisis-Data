# Proyek Analisis Data Penyewaan Sepeda

## Setup environment
```
conda create --name main-ds python=3.9
conda activate main-ds
pip install numpy

pip install pandas

pip install seaborn

pip install streamlit

pip install matplotlib

```

## Run streamlit app
```
streamlit run D:\UB\MBKM\Anindya-Project Akhir
```
